# たほいやのルール

このゲームは、Web上の「ルーム」に参加したプレイヤー同士で遊ぶ「たほいや」です。辞書に載っている珍しい単語の「本当の意味」と、他のプレイヤーが作った「偽の意味」を見分けることを目的とします。

## 1. ゲームの準備

*   参加人数: 1ルームあたり3〜8人です。
*   必要なもの:
    *   PC、スマートフォン、またはタブレット
    *   Webブラウザ
*   ゲーム開始前:
    *   プレイヤーは代表者が作成したルームに参加します。
    *   全員が揃ったら、ルームのホストがゲームを開始します。初期持ち点は全員10点でスタートします。

## 2. ゲームの流れ

ゲームは複数のラウンドで構成されます。参加しているプレイヤー全員が一度ずつ「親」を務めたらゲーム終了です。

### 1ラウンドの流れ

1.  親の選出:
    *   システムが自動でそのラウンドの「親」（出題者）を1人選びます。残りのプレイヤーは「子」となります。

2.  お題の提示:
    *   システムが辞書からランダムに珍しい単語を選び、親に提示します。
    *   お題はひらがなのみで出題されます。
    *   親はそのお題を確認し、全員に公開します。

3.  意味の入力:
    *   子: お題の単語に対する「もっともらしい偽の意味」をテキストボックスに入力し、送信します。
    *   親: システムに表示されている「本当の意味」をそのまま、あるいは少しだけ表現を整えて（てにをは等）入力し、送信します。

4.  選択肢の表示:
    *   全員の入力が終わると、システムが「本当の意味」と全員が作った「偽の意味」をシャッフルし、匿名で選択肢として画面に表示します。

5.  投票（ベット）:
    *   各「子」は、選択肢の中から「本当の意味」だと思うものに投票します。この時、持ち点の中からベットする点数を、1〜3点の間で同時に選択します。

6.  結果発表と点数計算:
    *   全員の投票が終わると、システムが正解と各選択肢の作者を発表します。
    *   以下のルールに基づいて点数が自動で計算され、各プレイヤーの持ち点に反映されます。

## 3. 点数計算

### 親の点数

*   子が「本当の意味」を当てた場合、親はその子に賭けられた点数分のチップを支払います。
*   子が「偽の意味」に騙された場合、親はその子から1点を獲得します。

### 子の点数

*   自分が正解した場合:
    *   親から賭けた点数分のチップを受け取ります。
*   自分が不正解だった場合:
    *   自分が投票した「偽の意味」の作者に、賭けた点数分のチップを支払います。
    *   さらに、親にも1点を支払います。
*   自分の作った「偽の意味」が他の子に選ばれた場合:
    *   その「偽の意味」を選んでくれた子から、その子が賭けた点数分のチップを受け取ります。

## 4. ゲームの終了と勝者

*   参加者全員が一度ずつ親を務めると、ゲームは終了します。
*   最終的に最も持ち点が高かったプレイヤーが勝者となります。
