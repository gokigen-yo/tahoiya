# 要求定義書

## アプリケーションの目的

本アプリケーションは、辞書にある珍しい単語の意味を当てるゲーム「たほいや」をWeb上で提供する。
Web上に広く公開し、多くの人が遊べるようにすることを目的とする。

## ターゲットユーザー

友人・知人同士のグループでの利用を想定する。

## 利用イメージ

1.  ユーザー（ホスト）がゲームをプレイするための「ルーム」を作成する。
2.  作成されたルームの専用URLが発行される。
3.  ホストは、一緒に遊びたい仲間をそのURLに招待する。
4.  招待された仲間はURLにアクセスし、ルームに参加してゲームをプレイする。

## 主要な方針

* プレイヤー認証: アカウント登録機能は設けず、プレイヤーはゲーム参加時に任意の名前を入力する。
* 対応デバイス: 主にPCのWebブラウザでの利用を想定する。スマートフォン対応は現時点では必須としない。
